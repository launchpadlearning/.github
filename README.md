## 🎯 Event Games

Many repositories here are lightweight web games used at live community events.

We intentionally use vanilla HTML, CSS, and JavaScript whenever possible:

- No installs  
- No build tools  
- No dependencies  
- Easy to understand  
- Easy to teach from  

Simple code. Real impact.
